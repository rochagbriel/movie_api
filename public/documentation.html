MyFlix - Application Documentation

The "MyFlix" application is a web app that provides users with access to information about different movies, directors, and genres.

---------------------

Endpoints

Return a list of ALL movies to the user

Description:

This endpoint returns a list of all available movies.

URL:

"/movies"

Method Type:

GET

Query Parameters(URL):

None

Request data format:

JSON Object

Response data format:

  • ID: Integer
  • Username: String
  • Password: String
  • Email: String
  • Birthday: Date
---------------------

Get data about a specific movie by title

Description:

This endpoint returns information about a specific movie, based on its title.

URL:

"/movies/:title"

Method Type:

GET

Query Parameters(URL):

title

Request data format:

JSON Object

Response data format:

JSON Object

---------------------

---------------------

Allow New Users to Register

Description:

Register a new user account.

URL:

"/users"

Method Type:

POST

Query Parameters(URL):

None

Request data format:

  • Username: String(required)
  • Password: String(required)
  • Email: String(required)
  • Birthday: Date(required)

Response data format:

Returns a JSON object with a success message and the ID of the newly created user.

---------------------

Allow Users to Update Their User Info

Description:

Update a user's data.

URL:

"/users/:id"

Method Type:

PUT

Query Parameters(URL):

id

Request data format:

JSON Object including a key you want to change

Sample:

{ Username: 'New User Name' }

Response data format:

Returns a JSON object with a success message.

---------------------

Allow Users to Add a Movie to Their List of Favorites

Description:

Add a movie to a user's list of favorites.

URL:

"/users/:id/movies/:MovieId"

Method Type:

POST

Query Parameters(URL):

id and movieTitle

Request data format:

id/movieTitle

Response data format:

Returns a JSON object with a success message.

---------------------

Allow Users to Remove a Movie from Their List of Favorites

Description:

Remove a movie from a user's list of favorites.

URL:

"/users/:id/:MovieId"

Method Type:

DELETE

Query Parameters(URL):

id and movieTitle

Request data format:

id/movieTitle

Response data format:

Returns a JSON object with a success message.

---------------------

Allow Existing Users to Deregister

Description:

Delete an existing user account.

URL:

"/users/:id"

Method Type:

DELETE

Query Parameters(URL):

id

Request data format:

id

Response data format:

Returns a success message.
