MyFlix - Application Documentation

The "MyFlix" application is a web app that provides users with access to information about different movies, directors, and genres.

---------------------

Endpoints

Return a list of ALL movies to the user

Description:

This endpoint returns a list of all available movies.

URL:

"/movies"

Method Type:

GET

Query Parameters(URL):

None

Request data format:

JSON Object

Response data format:

SAMPLE:

        {
            "Genre": {
                "Name": "Drama",
                "Description": "Drama is a category or genre of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone."
            },
            "Director": {
                "Name": "Jonathan Demme",
                "Bio": "Updated Director Bio",
                "Birth": "1944",
                "Death": "2017"
            },
            "Actors": [],
            "_id": "643d4c7abe406410ca35304e",
            "Title": "Philadelphia",
            "Description": "Updated Description Philadelphia",
            "ImagePath": "philadelphia.png",
            "Featured": false
        }

Get data about a specific movie by title

Description:

This endpoint returns information about a specific movie, based on its title.

URL:

"/movies/:title"

Method Type:

GET

Query Parameters(URL):

title

Request data format:

SAMPLE: http://localhost:8080/movies/Forrest%20Gump

Response data format:

Get data about a genre by name

Description:

This endpoint returns information about a specific genre, based on its name.

URL:

"/movies/genre/:GenreName"

Method Type:

GET

Query Parameters(URL):

genreName

Request data format:

SAMPLE: http://localhost:8080/movies/genre/Drama

Response data format:

SAMPLE:

        {
            "Name": "Drama",
            "Description": "Drama is a category or genre of narrative fiction (or semi-fiction) intended to be more serious than humorous in tone."
        }

Allow New Users to Register

Description:

Register a new user account.

URL:

"/users"

Method Type:

POST

Query Parameters(URL):

None

Request data format:

SAMPLE:

            {
                "Username": "testuser",
                "Password": "testpassword",
                "Email": "testuser@email.com",
                "Birthday": "09/02/53"
            }

Response data format:

SAMPLE:

            {
                "Username": "testuser",
                "Password": "testpassword",
                "Email": "testuser@email.com",
                "Birthday": "1953-09-02T00:00:00.000Z",
                "FavoriteMovies": [],
                "_id": "6443091a4ca183e11e04ae41",
                "__v": 0
            }
---------------------

Allow Users to Update Their User Info

Description:

Update a user's data.

URL:

"/users/:id"

Method Type:

PUT

Query Parameters(URL):

id

Request data format:

JSON Object including a key you want to change

Sample:

{ Username: 'New User Name' }

Response data format:

Returns a JSON object with a success message.

SAMPLE: 

            {
                "_id": "6443091a4ca183e11e04ae41",
                "Username": "testuserUpdated1",
                "Password": "testpassword",
                "Email": "testuser@email.com",
                "Birthday": "1953-09-01T23:00:00.000Z",
                "FavoriteMovies": [
                    "643da1febe406410ca353051"
                ],
                "__v": 0
            }
---------------------

Allow Users to Add a Movie to Their List of Favorites

Description:

Add a movie to a user's list of favorites.

URL:

"/users/:id/movies/:MovieId"

Method Type:

POST

Query Parameters(URL):

id and movieTitle

Request data format:

SAMPLE: http://localhost:8080/users/6443091a4ca183e11e04ae41/movies/643d4c7abe406410ca35304e

Response data format:

Returns a JSON object with a success message.

SAMPLE: 

            {
                "_id": "6443091a4ca183e11e04ae41",
                "Username": "testuserUpdated1",
                "Password": "testpassword",
                "Email": "testuser@email.com",
                "Birthday": "1953-09-01T23:00:00.000Z",
                "FavoriteMovies": [
                    "643da1febe406410ca353051",
                    "643d4c7abe406410ca35304e"
                ],
                "__v": 0
            }
